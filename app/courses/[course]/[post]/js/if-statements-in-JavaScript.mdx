## If, Else If, Else Statements

For handling decisions based on conditions, the `if`, `else if`, `else` statements are indispensable. For instance, consider displaying messages based on a customer's order size in a coffee shop scenario:

```javascript
let orderAmount = 16;

if (orderAmount > 15) {
    console.log("Enjoy your free cookie!");
} else if (orderAmount > 10) {
    console.log("Enjoy your free coffee!");
} else {
    console.log("Thank you for your purchase!");
}
```
