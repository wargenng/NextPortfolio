### Week 4: Document Object Model (DOM) Manipulation (90-minute class)

#### Introduction to the DOM (15 minutes)

-   **What is the DOM?** The DOM is a programming interface for web documents. It represents the page so programs can change the document structure, style, and content.
-   **Tree Structure:** The DOM is structured as a tree of objects such as elements, attributes, and text nodes.

#### Selecting Elements (10 minutes)

-   **Methods to Select Elements:**
    -   `getElementById`: `document.getElementById('example-id')`
    -   `querySelector`: `document.querySelector('.example-class')`
-   **Example:** Changing the text of an element with ID `example-id`.
    ```javascript
    document.getElementById("example-id").textContent = "New text content!";
    ```

#### Modifying Elements (20 minutes)

-   **Changing Content:**
    -   Change text content: `element.textContent = 'Hello World!';`
    -   Change HTML content: `element.innerHTML = '<strong>Hello World!</strong>';`
-   **Changing Styles:**
    -   `element.style.backgroundColor = 'yellow';`
-   **Attributes:**
    -   Set attribute: `element.setAttribute('href', 'https://example.com');`
-   **Example:** Making an element visible or hidden on button click.
    ```javascript
    document
        .querySelector("#toggle-button")
        .addEventListener("click", function () {
            const element = document.querySelector("#toggle-element");
            element.style.display =
                element.style.display == "none" ? "" : "none";
        });
    ```

#### Creating and Removing Elements (15 minutes)

-   **Creating Elements:** Use `document.createElement` and `parentNode.appendChild`.
    ```javascript
    const listItem = document.createElement("li");
    listItem.textContent = "New item";
    document.querySelector("ul").appendChild(listItem);
    ```
-   **Removing Elements:** Use `parentNode.removeChild`.
    ```javascript
    const firstItem = document.querySelector("li");
    firstItem.parentNode.removeChild(firstItem);
    ```

#### Handling Events (20 minutes)

-   **Introduction to Events:** Events like clicks, mouse movements, key presses allow interaction with the webpage.
-   **Adding Event Listeners:**
    -   `element.addEventListener('click', function() { alert('Clicked!'); });`
-   **Event Propagation:** Understanding event bubbling and capturing.
-   **Example:** Logging a message when a button is clicked.
    ```javascript
    document
        .querySelector("#log-button")
        .addEventListener("click", function () {
            console.log("Button clicked!");
        });
    ```

#### Practical DOM Manipulation Example (10 minutes)

-   **Interactive Web Page Example:** Building a dynamic list that users can add items to.
-   **Live Coding Example:** Creating a form that adds user input to a list.
    ```javascript
    document
        .querySelector("#submit-button")
        .addEventListener("click", function (event) {
            event.preventDefault(); // Prevent form submission
            const userInput = document.querySelector("#user-input").value;
            const listItem = document.createElement("li");
            listItem.textContent = userInput;
            document.querySelector("#item-list").appendChild(listItem);
        });
    ```
