## Practical Exercise

To reinforce what you've learned, try writing a function to calculate the area of a rectangle:

```javascript
function calculateArea(length, width) {
    return length * width;
}

console.log(calculateArea(10, 5)); // 50
```

Then, convert this function into an arrow function:

```javascript
const calculateArea = (length, width) => length * width;
console.log(calculateArea(10, 5)); // 50
```
