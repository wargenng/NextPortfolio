## Functions: Reusable Code Blocks

Functions allow you to encapsulate code for reuse, making your code more modular and manageable.

### Function Declarations

Declare functions to perform specific tasks, like calculating the total price after tax:

```javascript
function calculateTotalPrice(amount, taxRate) {
    return amount + amount * taxRate;
}

console.log(calculateTotalPrice(100, 0.15)); // 115
```
