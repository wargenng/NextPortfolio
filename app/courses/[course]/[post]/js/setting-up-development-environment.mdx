## 3. Setting Up a Development Environment

-   **Visual Studio Code (VS Code):** A versatile code editor.
    -   **Installation:** [Visual Studio Code](https://code.visualstudio.com/)
    -   **Getting Started Guide:** [VS Code Documentation](https://code.visualstudio.com/docs)
-   **Node.js:** Enables JavaScript to run outside the browser.
    -   **Installation:** [Node.js](https://nodejs.org/), choose the LTS version.
    -   **Getting Started:** [Node.js Documentation](https://nodejs.org/en/docs/)
