**Basic Arithmetic Operations**

-   **Explanation**: Discuss how to perform arithmetic calculations in Python.
-   **Operations**:
    -   Addition (`+`)
    -   Subtraction (`-`)
    -   Multiplication (`*`)
    -   Division (`/`)
    -   Floor division (`//`)
    -   Modulus (`%`)
    -   Exponentiation (`**`)
-   **Examples**:
    ```python
    print(10 + 5)  # Addition
    print(10 - 5)  # Subtraction
    print(10 * 5)  # Multiplication
    print(10 / 5)  # Division
    print(10 // 3)  # Floor Division
    print(10 % 3)   # Modulus
    print(10 ** 2)  # Exponentiation
    ```
