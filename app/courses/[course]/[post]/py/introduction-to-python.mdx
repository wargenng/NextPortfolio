### Your First Python Script

#### Hello World

Start your programming journey with a simple script that outputs "Hello, World!".

```python
# This line is for humans - it's a comment and ignored by Python
print("Hello, World!")
```

-   `print()` displays text on the screen.

#### Variables and Data Types

Variables store data that we can manipulate. Python includes several data types:

-   **Integers**: Whole numbers.
-   **Floats**: Decimal numbers.
-   **Strings**: Text.

```python
age = 25  # Integer
temperature = 98.6  # Float
name = "Charlie"  # String
print("Name:", name, "Age:", age, "Temperature:", temperature)
```

### String Operations

Combine strings and variables to create messages with dynamic data.

```python
# Using + to combine strings
greeting = "Hi, " + name + "! You're " + str(age) + " years old."
print(greeting)

# Using format() for cleaner code
greeting = "Hi, {}! You're {} years old.".format(name, age)
print(greeting)
```

### Mini Challenge: Personalized Greeting

Create a script that personalizes a greeting based on user input, including doubling their favorite number.

```python
user_name = input("Your name: ")
favorite_number = int(input("Favorite number: "))
message = "Hello, {}! Twice your favorite number is {}.".format(user_name, favorite_number * 2)
print(message)
```
